Maven学习
--------------------
mvn生命周期的命令：
 #. mvn clean：清理编译的项目
 #. mvn compile：编译项目
 #. mvn test：测试项目（运行测试类）
 #. mvn package：负责将我们的项目打包
 #. mvn install：将这个项目安装到仓库中
